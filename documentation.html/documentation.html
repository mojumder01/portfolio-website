👋 Introduction

Thank you for purchasing the Alex Morgan Portfolio Template! This is a fully-featured, single-page portfolio template built with pure HTML5, CSS3, and vanilla JavaScript — no frameworks, no build tools, no npm required.

You can open it directly in any browser, upload it to any web host, or deploy it for free on GitHub Pages, Netlify, or Vercel.

💡

This template is designed to be customizable without any coding knowledge. Every section that needs your content is marked with a CHANGE: comment in the HTML and JS files.

What's Included

  • index.html — Main template file
  • css/style.css — All styles with 5 color schemes
  • js/script.js — All interactions and animations
  • documentation.html — This file
  • assets/ — Folder for your photo and CV

Features at a Glance

  • 🎨 5 built-in color schemes (Purple, Blue, Green, Orange, Red)
  • 🌙 Dark / Light mode toggle with localStorage persistence
  • ⏳ Animated preloader
  • ✍️ Typing animation (fully customizable text)
  • ✨ Particle canvas background
  • 🖱️ Custom cursor (desktop)
  • 🛠️ Services section with 6 cards
  • 📅 Experience timeline with Work / Education tabs
  • 🔍 Projects section with filter (All / Web / Mobile / Design)
  • 💬 Testimonials slider with auto-play
  • 🔢 Animated counter stats
  • 📬 Contact form (opens email client)
  • 📱 Fully responsive — mobile, tablet, desktop
  • ♿ Accessible — semantic HTML, aria-labels

🚀 Installation

No build process needed. Just open the files and go.

1

Extract the ZIP

Unzip the downloaded file into any folder on your computer.

2

Open in browser or editor

Double-click index.html to preview in your browser. Or open the folder in VS Code for editing.

3

Customize your content

Follow the sections below to update your name, photo, links, skills, and projects.

4

Deploy

Upload all files to your web host, or push to GitHub and enable GitHub Pages for a free live site.

Deploy to GitHub Pages (Free)

  1. Create a new GitHub repository (e.g. my-portfolio)
  2. Upload all template files to the repository
  3. Go to Settings → Pages → Source: main branch → / (root)
  4. Your site will be live at https://yourusername.github.io/my-portfolio

📸 Profile Photo

The hero section currently shows a placeholder with your initials. To use a real photo:

1

Prepare your photo

Use a square photo, minimum 400×400px. JPG or PNG format. Name it photo.jpg.

2

Place it in the assets folder

Copy your photo to assets/photo.jpg.

3

Update the HTML

In index.html, find the hero photo section and replace the placeholder:

<!-- Remove this line: -->
<div class="hero-photo-placeholder">AM</div>

<!-- Add this line instead: -->
<img src="assets/photo.jpg" alt="Your Name" />
💡

For the best look, use a photo with a clean or transparent background. The template applies a circular crop automatically.

The same placeholder exists in the About section avatar. Update it similarly:

<!-- About section avatar -->
<div class="avatar-initials">AM</div>
<!-- Replace with: -->
<img src="assets/photo.jpg" alt="Your Name" style="width:100%; height:100%; object-fit:cover; border-radius:50%;" />

🎨 Color Schemes

The template includes 5 built-in accent colors. The color switcher panel (🎨 button on the right side) lets visitors switch colors live. The chosen color is saved in localStorage.

Purple Default
Blue
Green
Orange
Red

Change the Default Color

To change what color loads by default, update the data-color attribute on the <html> tag:

<!-- Change "purple" to any of: blue, green, orange, red -->
<html lang="en" data-theme="dark" data-color="purple">

Add a Custom Color

Open css/style.css and add a new block after the existing color schemes:

/* Your custom color */
:root[data-color="custom"] {
  --primary:     #your-color;
  --primary-dim: rgba(r, g, b, 0.15);
  --accent:      #your-accent;
  --glow:        rgba(r, g, b, 0.45);
  --card-hover:  rgba(r, g, b, 0.08);
  --tag-bg:      rgba(r, g, b, 0.15);
  --tag-color:   #lighter-shade;
  --particle:    r, g, b;   /* RGB values, no rgba() */
}

Then add a dot button in the HTML color switcher panel:

<button class="cdot" data-color="custom" title="Custom"></button>

And the dot color in CSS:

.cdot[data-color="custom"] { background: #your-color; }

Change Default Theme (Dark/Light)

<!-- Change "dark" to "light" for light mode default -->
<html lang="en" data-theme="light" data-color="purple">

💬 Testimonials

Testimonials are inside the #testimonialsTrack div. Each card is a .testimonial-card.

If you add or remove testimonial cards, also update the slider dots in the HTML and the total variable in js/script.js is calculated automatically from the dot count.

<div class="testimonial-card">
  <div class="testimonial-stars">★★★★★</div>
  <p class="testimonial-text">"Your client's review..."</p>  <!-- CHANGE -->
  <div class="testimonial-author">
    <div class="author-avatar">AB</div>   <!-- CHANGE: initials -->
    <div>
      <strong>Client Name</strong>          <!-- CHANGE -->
      <span>Role, Company</span>            <!-- CHANGE -->
    </div>
  </div>
</div>

Also add a matching dot for each extra card:

<button class="slider-dot" data-index="3"></button>

📬 Contact Form

The contact form uses mailto: — when submitted, it opens the visitor's default email client with a pre-filled message. No backend or server is required.

Update Your Email Address

Open js/script.js and find the contact form section (around line 230):

// CHANGE: Replace with your email address
window.location.href = `mailto:hello@alexmorgan.dev?subject=...`;

Replace hello@alexmorgan.dev with your actual email.

ℹ️

For a real contact form that sends emails without opening a mail client, consider using Formspree (free plan available). Just replace the form's action URL.

Using Formspree (Optional)

<!-- Replace the form tag to use Formspree -->
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
